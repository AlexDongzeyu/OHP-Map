Crestwood Oral History Project · 0 journeys

A student project that adapts the open-source HandsOnDataViz Leaflet Storymaps idea for the guided view, and adds a custom Explore / Patterns / time-scrubber app. Built on Leaflet (BSD-2-Clause) with CARTO basemaps. Source of truth: ohp.crestwood.on.ca. Made with restraint, in memory of those whose journeys these are.
